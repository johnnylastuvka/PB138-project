Záverečná správa o mojej práci na projekte

Projekt: Webová aplikácia pre správu videotéky

Autor: Jozef Mikušinec

Ako vedúci projektu som založil a spravoval GitHub repositár, koordinoval tímovú spoluprácu, a rozdelil som projekt na viacero podúloh, ktoré sme si v tíme rozdelili medzi seba.

Takisto som vytvoril "kostru" projektu - základ repozitára, a vytvoril som základný backend.

Podúlohy

Projekt bol rozdelený do štyroch hlavných podúloh, a to:

  1. Lead developer, backend (ja)
  2. Backend (Ondrej Novák)
  3. User interface design (Simona Nemčoková)
  4. User interface functionality (Jan Laštůvka)

Okrem týchto úloh sme ešte dynamicky riešili a prideľovali si ostatné záležitosti, ako napríklad tvorbu dokumentácie.

Backend, kostra projektu

Vytvoril som základnú kostru projektu a časť backendu. To zahŕňa spustenie serveru, spracovanie http požiadavkov a ich predanie triede LibraryEditor, ktorá požiadavky vykonáva, a ktorá bola implementovaná Ondrejom Novákom.
